# 12. Glossary

| Term                 | Definition                                                                                                                       |
| -------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| HTTP                 | [Hypertext Transfer Protocol](https://developer.mozilla.org/en-US/docs/Web/HTTP)                                                 |
| API                  | [Application Programming Interface](https://en.wikipedia.org/wiki/API)                                                           |
| RFC                  | [Request for Comments](https://www.ietf.org/process/rfcs/)                                                                       |
| Developer            | A person who writes software, including APIs. This person uses Thymian and may be interested in writing plugins or rules for it. |
| Thymian Proxy Plugin | Proxy plugins (there can be more than one) connect a 3rd party software or external plugin to Thymian.                           |
| Rule                 | A definition that is used by Thymian for static or dynamic API verification.                                                     |
