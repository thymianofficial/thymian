- to include new rules just provide the name/path of the rule (set) to the `--rules` flag or via the thymian configuration
- for an example rule set see the `rfc-9110-rule` project
- to publish a rule set just publish your set as npm package and include the name in the thymian config or via the cli. Note that the package must be installed in a first step.
- if a rules required some configuration this can be also done via the thymian config.
