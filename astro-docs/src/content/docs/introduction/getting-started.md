title: Getting Started
description: Getting Started with Thymian
---

Let's first install Thymian. Since Thymian is still in development, you'll need to clone the repository and install the dependencies.

```bash
git clone git@github.com:thymianofficial/thymian.git
cd thymian
npm install
nx build thymian
```

To use the Thymian CLI you must navigate into the `packages/thymian` directory. From there, you can run Thymian commands using `./bin/run.js <command>`.
