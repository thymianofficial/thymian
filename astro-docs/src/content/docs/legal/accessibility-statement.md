## Service Implementation Details

Through thymian.dev, we provide publicly accessible and free documentation for the Open Source Software Thymian.

## Compliance Status with Requirements

The above-mentioned service is fully compliant with the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA and the Barrier-Free Strengthening Act (BFSG).

The following measures were taken to ensure accessibility:

- All text meets WCAG AA contrast ratios (4.5:1 for normal text, 3:1 for large text)
- All interactive elements are keyboard accessible with visible focus indicators
- Content remains visible and usable when JavaScript is unavailable
- Responsive layouts use correct semantic markup for assistive technologies
- Dark mode theming respects the Starlight theme toggle rather than relying solely on OS-level preference
- Automated accessibility testing using axe-core is part of the continuous integration pipeline
